Smart Classroom AI

Smart Classroom AI is a system that uses Artificial Intelligence to monitor student activities in the classroom and provide insights through a mobile application.

The system detects student activity using AI and sends the results to a backend server. The backend stores the data in a database and provides APIs for a Flutter mobile app to display the information.

Project Objective

The goal of this project is to create a smart classroom monitoring system that helps teachers understand student engagement and classroom activity using AI technology.

Technologies Used
	•	Python (AI processing)
	•	Flask (Backend API)
	•	SQLite (Database)
	•	Flutter (Mobile application)
	•	Git & GitHub for version control

System Architecture

The system works in the following flow:

Camera / Input
↓
AI Activity Detection (Python)
↓
Flask API Server
↓
SQLite Database
↓
Flutter Mobile App
